# WDD231 - Web Frontend Development I

This repository contains materials and projects for the **WDD231 - Web Frontend Development I** course, created by **Jose D. Colmenarez Gil**. This course is part of the curriculum at **BYU-Idaho** and serves as the final requirement for earning the first certificate in the **Applied Technology** program.

## Course Overview

WDD231 focuses on foundational skills in web frontend development, including:

- HTML5 and semantic structure
- CSS3 for styling and layout
- Responsive design principles
- Introduction to JavaScript for interactivity

## Purpose

This course represents a significant milestone in the journey toward mastering web development and achieving the first certificate in the Applied Technology program.
