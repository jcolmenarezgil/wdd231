Site Plan | WDD 321 Final Project

Site Name

Ucaima Import

This name represents an existing company in Ecuador, which wishes to incorporate e-commerce through its own website.

Site Purpose

For a Small Entrepreneur in Quito, Ecuador.

The website offers a product catalog and a user-friendly interface allowing users to filter content by categories for greater interest. It also provides recommendations based on previously viewed products and includes a form next to the shopping cart to simulate product purchase.

Scenarios

Questions that users might ask

  • Where can I find more details about the items I am interested in (size, colors available, etc)?
  • Can I filter the content to better focus on what I am looking for?
  • How can I place an order online?
